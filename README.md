🌿 MedTrack – Your Health, Our Priority
💡 Built with care by Mounika – simplifying healthcare through technology.

MedTrack is a cloud-based healthcare management system developed using Flask and AWS DynamoDB. It offers a seamless platform for doctors and patients to connect, manage appointments, upload diagnostic reports, and access medical histories securely — all from your browser.

✨ Key Highlights
🔐 Secure Role-Based Login – Separate dashboards for doctors and patients.

🩺 Book Appointments Easily – Schedule visits without waiting in line.

📁 Upload Reports Instantly – Doctors can submit diagnoses & files online.

🧾 Track Medical History – Patients can review past consultations and reports.

⚡ Powered by AWS DynamoDB – Fast, reliable NoSQL cloud storage.

🛠️ Tech Stack
Frontend: HTML5, CSS3, Bootstrap

Backend: Python (Flask)

Database: AWS DynamoDB

Cloud Storage (optional): AWS S3

Session Management: Flask Sessions
